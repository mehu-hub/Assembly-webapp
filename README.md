# 🖥️ PC Assembly Manager

> A web-based CRUD application for managing PC parts, component inventory, stock tracking, and product assembly calculations.

---

## 📋 About the Project

PC Assembly Manager helps track which components are needed to assemble PC products. It allows users to manage product structures, monitor component stock across the workshop and storage, and calculate how many products can be assembled based on available inventory.

---

## ✨ Features

### 🔵 Products
- View all products in a product list
- Define product structure (components + quantities)
- Search product by part of name
- Manage product assembly with component dropdown and quantity validation

### 🟣 Components
- View full component list
- Track components in workshop and in storage separately
- View component inventory report

### 🟢 Stock
- Manage component stock with price and quantity
- View stock quantities and component prices

### 🟡 Assembly
- See required components for each product
- Calculate how many products can be assembled
- Use the Assembly Calculator based on current stock

### 🩷 Reports
- Product Structure Report
- Component Inventory Report
- Assembly Possibility Report
